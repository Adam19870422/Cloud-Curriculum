1. Run the tests:

    === "Command Line"
        ```shell
        mvn test
        ```
    === "IntelliJ"
        - Right-click on the project -> `Run All Tests`
    === "Spring Tool Suite"
        - Right-click on the `src>test>java` directory and click Run 'All Tests'.
    === "Visual Studio Code"
        - Right-click on the `src>test>java` directory and click `Run Tests`.
